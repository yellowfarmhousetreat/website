## Reporting a Vulnerability

If you discover a security vulnerability in our website, please report it responsibly:

### For Security Issues

- **Email:** [Create an issue in this repository]
- **Response Time:** We aim to respond within 48 hours
- **Disclosure:** Please allow us time to address the issue before public disclosure

### What to Include

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact assessment
- Suggested fix (if available)
